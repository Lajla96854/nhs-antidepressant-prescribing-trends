## NHS Antidepressant Prescribing Trends

This project analysis correlation between NHS antidepressant prescriptions and its costs in England.

Data
- Prescription Cost Analysis (PCA), 2019–2024 `Prescription Cost Analysis (PCA) from NHS Digital.csv`
- Antidepressant prescriptions 2012–2024  `Antidepressant Prescriptions in England (2012–2024).png`
- NHS Cost 2019–2024  `NHS Cost for Antidepressant Drugs (2019–2024).png`

Reflection
- Prescriptions dispense increased from 2012–2024.
- NHS spend (2019–2024) doesnt reflect on the amount of prescribed drugs
- Cost per item varies throughout the analysed period but it is declining.
